# 头结构介绍
示例文件：main.c
```C
#include <unistd.h>
#include <stdio.h>

int main(int argc, char* argv[]){
  printf("this is elf test");
  return 0;
}
```

* .o：c/.c++文件编译出的obj文件
* .so：动态库文件
* .out：可执行程序

```shell
gcc -c main.c -o main.o
gcc main.c -o main.out
gcc -fPIC -shared main.c -0 main.so

readelf -h main.so
```
